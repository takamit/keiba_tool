# 競馬予想ツール 更新版

## できること
- 指定日の出走表取得
- 指定日の結果取得
- 過去結果CSVを学習用フォーマットに蓄積
- `target_top3` または `target_win` を目的変数として機械学習
- 学習済みモデルで当日レースを予測
- GUIから取得・学習・予測を操作

## フォルダ
- `data/entry_YYYYMMDD.csv` : 出走表
- `data/result_YYYYMMDD.csv` : 結果
- `data/predict_YYYYMMDD.csv` : 予測結果
- `models/*.joblib` : 学習済みモデル
- `cache/` : 取得HTMLキャッシュ
- `logs/app.log` : ログ

## 注意
- スクレイピング先のHTML構造が変わると、パーサ修正が必要です。
- 機械学習の精度は、集めた過去データ量と特徴量設計に強く依存します。
- まずは動く土台を作り、その後に血統・騎手成績・脚質・コース相性などを追加する設計です。
